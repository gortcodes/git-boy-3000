      HOW IT WORKS

      Public GitHub profile, public events (last 30), GraphQL contribution calendar (1 year), and — for the configured owner — a full walk of non-fork public repos for file-tree probing (terraform, helm, docker, CI/CD configs, observability configs, AI co-author trailers). A pure versioned engine converts those signals into SPECIAL stats and sub-skills. Redis cache with stale-while-revalidate, OpenTelemetry tracing, Prometheus metrics, Loki logs, Grafana dashboard. All runs in docker compose on one machine.
