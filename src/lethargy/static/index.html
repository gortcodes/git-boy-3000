How it works

        

          The service pulls three public GitHub sources (profile, public events, GraphQL
          contribution calendar) through an ETag-aware collector, extracts signals with a
          pure versioned engine, and scores six stats. A Redis-backed cache serves repeat
          lookups with stale-while-revalidate; owner snapshots persist to Postgres for
          history and replay. Everything is traced with OpenTelemetry and observed through
          Prometheus metrics and a Grafana dashboard.
